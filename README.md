# ATS Analyzer

AI-Powered Resume Analyzer and Job Matcher

## Project Overview

ATS Analyzer is a comprehensive application that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS). The application analyzes resumes against job descriptions, provides scoring, and offers optimization suggestions.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion
- React Router

## Getting Started

### Prerequisites

- Node.js & npm installed
- Bun (optional, for faster package management)

### Installation

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

### Building for Production

```sh
npm run build
```

## Features

- Resume Analysis against Job Descriptions
- ATS Score Calculation
- Resume Optimization Suggestions
- Job Search Integration
- Report History
- Multiple Resume Templates

## API Integration

The frontend communicates with the backend API. Configure the API endpoint in the `.env` file.
